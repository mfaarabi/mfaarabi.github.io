My Terminal Setup

2020-07-05 :: Farabi
#terminal  #personalization 

As a software engineer, I find myself using terminal as the de facto way to work. It offers a highly customizable text based interface that would otherwise be complicated if it’s a UI based interface. How it helps me to be productive depends on how I configure it. So here I present my own terminal setup.

I’ll go through what I install (and later I’ll explain its functionality).

ZSH⌗

Install ZSH⌗ 

sudo apt install zsh

Set WSL to run ZSH by default⌗

.bashrc first line

if [ -t 1 ]; then
  exec zsh
fi

oh-my-zsh⌗ 

sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
git clone https://github.com/denysdovhan/spaceship-prompt.git "$ZSH_CUSTOM/themes/spaceship-prompt"
ln -s "$ZSH_CUSTOM/themes/spaceship-prompt/spaceship.zsh-theme" "$ZSH_CUSTOM/themes/spaceship.zsh-theme"

Brew⌗ 

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
sudo apt install build-essential
echo 'eval $(/home/linuxbrew/.linuxbrew/bin/brew shellenv)' >> /home/farabi/.zshrc
eval $(/home/linuxbrew/.linuxbrew/bin/brew shellenv)
brew install gcc

The Rest⌗ 

brew install fzf
$(brew --prefix)/opt/fzf/install
brew install z
brew install tig
git clone https://github.com/lukechilds/zsh-nvm ~/.oh-my-zsh/custom/plugins/zsh-nvm
nvm install --lts
brew install yarn
brew install hugo
